    QVEX Lynepad:
    Macro keypad with 11 keys, a rotary joystick, and a thumb wheel.

    Description

    QVEX Lynepad is an additional, fully programmable keyboard suitable for any kind of design work including photo editing, video editing, PCB design, 3D modeling and so much more. It allows you to press shortcuts or run complex macros using a single key.

    Features

    • 11 Gateron MX green keys (80g force, clicky) (hot-swappable in rev.2.0 or newer)
    • 5-axis wheel (rotation and 4-way tilt + push)
    • Thumb wheel (rotation + push)
    • Ergonomic and minimalistic design
    • White backlighting (rev.2.0 or newer)
    • 4 programmable RGB LEDs
    • Power indication LED
    • Anti-slip silicone pads (only available with the enclosure option).
    • Prepared mapping for KiCad
    • A total of 21 physical actions (pressing button, rotating wheel etc.) and up to 54 actions using combinations.
